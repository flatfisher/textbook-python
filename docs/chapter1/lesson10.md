#タプルを使ってみる
タプルはPythonで扱える配列のようなものです。リストとの違いは、後から変更（追加や削除）ができないことです。(Tuple1.py)

```
value = (90, 80, 29)

print(value[0])
print(value[1])
print(value[2])
```

文字列の場合(Tuple2.py)

```
value = ("Hokkaido", "Aomori", "Akita")

print(value[0])
print(value[1])
print(value[2])
```
