#Print文を学ぶ
* * * * *
先ほど ```Hello World``` という文字列を表示させました。こちらはpythonのprintと呼ばれる関数を使用したことによって
画面に文字を出力することが出来ました。printは文字や画像を印刷するといった意味が含まれています。

```
print("Hello World")
```

結果

```
Hello World
```
